2. Installation

There are multiple ways to install Claude Code. Choose the method that works for you:

macOS Installation

Method 1: Using Curl (Recommended)

curl -fsSL https://claude.ai/install.sh | bash

Method 2: Using Homebrew

brew install --cask claude-code

Note: Open Terminal app (found in Applications → Utilities) to run these commands.

Windows Installation

Method 1: Using PowerShell (Recommended)

irm https://claude.ai/install.ps1 | iex

Method 2: Using WinGet

winget install Anthropic.ClaudeCode

Note: Open PowerShell (search for it in Start menu) to run these commands.

Linux Installation

Using Curl

curl -fsSL https://claude.ai/install.sh | bash

Supported: Ubuntu 20.04+, Debian 10+, Alpine 3.19+

Verify Installation

After installation, verify Claude Code is installed correctly:

claude --version

You should see the version number displayed.

3. VS Code Extension Setup

Install the Claude Code extension in VS Code for the best experience:

1

Open VS Code Extensions

Press Cmd+Shift+X (Mac) or Ctrl+Shift+X (Windows/Linux)

Or click the Extensions icon in the left sidebar

2

Search for Claude Code

Type "Claude Code" in the search box

Look for the official extension from Anthropic

3

Install and Authenticate

Click "Install" button

Follow the browser prompts to authenticate with your Claude account

4

Access Claude Code

Click the Claude icon in the VS Code sidebar

Or press Cmd/Ctrl + Shift + P and type "Claude"

You're Ready!

The VS Code extension uses the same configuration as the CLI. Everything syncs automatically.

4. Understanding the Terminal

The terminal is where Claude Code runs. Don't worry - you don't need to be an expert!

What is the Terminal?

The terminal is a text-based interface where you can:

  • Type commands (like "claude") to run programs
  • Navigate folders on your computer
  • See Claude Code's responses

Basic Terminal Commands (All You Need)

pwd

Print Working Directory - Shows where you are

ls (Mac/Linux) or dir (Windows)

List files in current folder

cd foldername

Change Directory - Move to a folder

cd ..

Go up one folder level

mkdir foldername

Make Directory - Create a new folder

claude

Start Claude Code!

Pro Tip: Create a Practice Folder

mkdir ~/playground
cd ~/playground
claude

This creates a safe space to experiment without affecting important files!

5. Your First Task

Let's try something simple to see Claude Code in action:

1

Start Claude Code

claude

In VS Code, click the Claude icon or use Command Palette

2

Try This First Command

List all files in this folder and tell me what you find

Claude will search your directory and explain what it sees!

3

Watch Claude Work

You'll see Claude use "tools" like:

  • Bash - Running the ls/dir command
  • Read - Reading file contents

This is Claude acting, not just chatting!

4

Try Another Task

Claude will create the file for you. Check your folder - it's really there!

🎉 Congratulations!

You just used AI to interact with your computer. That's the core of Claude Code!

6. Understanding File Access

One of Claude Code's superpowers is direct file access. Here's what that means:

Traditional AI Chat

  1. You copy file content
  2. Paste into chat
  3. Get response
  4. Copy response
  5. Paste back into file
  6. Save file

Claude Code

  1. Tell Claude what to do
  2. Claude reads the file
  3. Claude edits the file
  4. Done!

What Claude Can Access

  • Text files - Code, documents, configuration files
  • Images - Screenshots, diagrams, photos (Claude can read text from images!)
  • PDFs - Documents and manuals
  • Folders - Entire project structures

Example: Editing a Document

Make my proposal.docx more professional and concise. Focus on the executive summary section.

Claude will read the file, understand the content, make improvements, and save it - all automatically!

Permission System

Don't worry about Claude making unwanted changes. You'll be asked to approve:

  • First time running any command
  • Every file modification
  • Any potentially dangerous operations

You're always in control!
