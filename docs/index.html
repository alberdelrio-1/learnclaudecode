What is Claude Code?

Claude Code is an AI-powered coding assistant that helps you with software development tasks. Think of it as a highly skilled colleague who:

  • Reads your files - Understands your codebase without you explaining it
  • Edits code directly - Makes changes to files on your computer
  • Runs commands - Executes terminal commands and builds projects
  • Learns from context - Remembers your preferences and project standards
  • Works with your tools - Integrates with VS Code, GitHub, Salesforce, and more

For Non-Coders: You don't need to know how to code to use Claude Code! It understands plain English instructions and can help you with tasks like editing documents, analyzing data, and automating workflows.

Your First Quick Win (5 Minutes)

Let's get you started with a simple task that shows Claude Code's power:

1

Open Claude Code in VS Code

Click the Claude icon in your VS Code sidebar or press Cmd/Ctrl + Shift + P and type "Claude"

2

Try Your First Command

Type: "Show me what files are in this project"

3

Watch Claude Work

Claude will list your files and explain what it found. You just used AI to explore your project!
