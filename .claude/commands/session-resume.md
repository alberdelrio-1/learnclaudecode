# Session Resume

This is a fresh session. Let's load the context from the previous session efficiently.

## Steps

1. **Read the handoff document**
   - Check `START-HERE-NEXT-SESSION.md` first
   - If not found, check the most recent entry in `SESSION-NOTES.md`

2. **Verify context**
   - Confirm the handoff is for the work you want to continue
   - Note the date and scope of the previous session

3. **Summarize for me**
   - What was completed
   - What's next on the priority list
   - Any critical warnings or context I should be aware of

4. **Ask clarifying questions**
   - Is there anything from the handoff that needs clarification?
   - Should we proceed with the next steps as planned, or adjust priorities?

5. **Begin work**
   - Once confirmed, start with the highest priority task from "Next Steps"

Keep your summary concise (under 150 words) to preserve context budget.
